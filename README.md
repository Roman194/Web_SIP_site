# Web_SIP_site
## Description
It's my plain website that was made during 6 labs as a practical task for the web-development course as part of the bachelor learning in the OmSTU (09.03.04 "Software programming").</br>
The task was to made an artists fan-site. It should have bio, announces and achievements sections. It also should have admin panel with CRUD operations and DB server as the match of two frontend-websites. 
Admin-panel should have an authorization page which can provide auth through social network
## Used features
- html + css (as a static websites markup) </br>
- php (as a way to connect to the DB and provide authorization, sessions and CRUD-operations)</br>
- js (as a dynamic websites content; used only in admin-panel)</br>
- XAMP (as a localhost with DB "MySQL") </br>
- CloudPub (as a globalhost; to provide auth through "VK") </br>
